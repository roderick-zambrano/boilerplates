## Setup

Set up the docker compose as provided in the sample. To enable simple authentication, use the generate command to create a user configuration meant to be stored in /data/users.yml, in the following form:

```
docker run -it --rm amir20/dozzle generate admin --password PASSWORD --email email@example.com --name Optional
```

where "admin" is the name of the user for login purposes.
